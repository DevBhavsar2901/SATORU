# 🌀 SAT0RU – Cursed Technique Visualizer

> “Throughout Heaven and Earth, I alone am the honored one.” ⚡

An immersive *Jujutsu Kaisen-inspired web experience* that uses *hand tracking + 3D particles* to simulate cursed techniques in real time.

---

## ✨ Features

* 🖐️ Real-time *Hand Tracking (MediaPipe)*
* 🌌 Stunning *3D Particle System (Three.js)*
* 🔥 Multiple Cursed Techniques:

  * 🔴 Reverse Cursed Technique: *Red*
  * 🟣 *Hollow Purple*
  * 🔵 *Infinite Void*
  * 🏯 *Malevolent Shrine*
* 💥 Smooth animations with bloom effects
* 🎥 Live camera interaction

---

## 🎮 How It Works

Different hand gestures trigger different cursed techniques:

| Gesture          | Technique            |
| ---------------- | -------------------- |
| ✊ Pinch          | 🟣 Hollow Purple     |
| ✋ All fingers up | 🏯 Malevolent Shrine |
| ✌️ Two fingers   | 🔵 Infinite Void     |
| ☝️ One finger    | 🔴 Red               |

---

## 🛠️ Tech Stack

* HTML, CSS, JavaScript
* Three.js (3D Graphics)
* MediaPipe Hands (Hand Tracking)
* WebGL + Post Processing

## 🚀 Run Locally (IMPORTANT)

⚠️ *Camera will NOT work if you open file directly (file://)*

### ✅ Correct way:

bash
git clone https://github.com/DevBhavsar2901/SATORU.git
cd SATORU
python3 -m http.server 8000


Now open in browser:


http://localhost:8000


👉 Allow camera access when prompted

## ⚠️ Common Issues

### ❌ Camera Not Working?

* Do NOT open using file://
* Use localhost server
* Allow camera permissions in browser

### 🍎 Mac Users

* Go to:
  System Settings → Privacy & Security → Camera
* Enable access for your browser
